# 简介

`vscode`配置`wsl`

# 流程

## 下载安装`vscode`:

[Download Visual Studio Code - Mac, Linux, Windows](https://code.visualstudio.com/Download)

安装`win`版,一直下一步,注意改一下安装位置

## 连接`wsl`

`ctrl + shift + x`打开应用商店,输入`Remote - WSL`安装由于事先安装好`wsl`了

打开左侧小电脑图标,点击进入即可进入`wsl`终端,可以通过点击左下角⚪,▲进入终端命令行
